1 Introduction

If you use emacs org mode to stay organized, sooner or later, there will be a requirement to publish your org files to html.

In html you can view the notes in your browser, deploy them to the web and share the site with anyone not into org mode.

2 Requirements

  • use emacs
  • use org mode > version 8.0 To check, M-x org-version Emacs 24 should come with org mode 8
  • convert org mode files to html

3 How to

3.1 Clone the org-html-themes repository

Link: https://github.com/fniessen/org-html-themes

3.2 Create an org file with the setup line on top

#+SETUPFILE: /path/to/org-html-themes/setup/theme-readtheorg.setup
#+TITLE: Title of my Org file

3.3 Export org file to html

M-x org-html-export-to-html
  • A html file will be created in the same directory as the org file
  • The html file should be viewable offline

3.4 Deploy the HTML file to any webserver
